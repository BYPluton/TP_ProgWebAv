Anime List : site web répertoriant des animes / series / films  

Utilisation possibles : 
Utilisateur : 
 - créer des watchlists 
 - communiquer sur la page de sa série/anime/film (une sorte de forum) 
 - noter une série/anime/film
 - suggérer l'ajout d'un série/anime/film

Modérateur : 
 - modérer une page 
 - ajouter des série/anime/film 

Pensez à comment intégrer les images : 
- url image internet stocker dans la base de données 
- nom image dans fichier et stocker dans base de données


Pages disponibles : 
- page de connection/inscription -> se connecter ou s'inscrire 
	-> nom, prenom, pseudo, mdp, mail (api envoie mail auto) 
- page d'accueil 
	-> aperçu des watchlists, création watchlist
- pages des différentes catégories (anime/serie/film) 
- page de profil  
	-> informations de l'utilisateurs
